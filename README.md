I have been working for a few hours on a small web app called SõnaSõber. It is designed to help learners practice Estonian sentence structures, grammar cases, and conjunctions using an interactive staging canvas.
Because it is a static web app, you do not need to install anything—it runs as a standard HTML file that opens directly in any web browser.
I would really appreciate your reviews, feedback, and any suggestions on how to make it more useful for language practice!
